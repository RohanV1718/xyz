# Modern React To-Do List

A sleek, visually rich To-Do List application built with React, TypeScript, and Vanilla CSS. This project focuses on a modern "Glassmorphism" aesthetic, providing a smooth and responsive user experience for managing daily tasks.

## 🚀 Features

-   **Add Tasks:** Quickly add new items to your list.
-   **Toggle Completion:** Mark tasks as done with a single click.
-   **Delete Tasks:** Remove tasks you no longer need.
-   **Task Filtering:** Filter your view by 'All', 'Active', or 'Completed' tasks.
-   **Data Persistence:** Your tasks are saved to `localStorage`, so they stay there even after a page refresh.
-   **Modern UI:** A beautiful, responsive design featuring glassmorphism effects, smooth transitions, and a clean layout.

## 🛠️ Tech Stack

-   **Frontend:** [React](https://reactjs.org/) (with [Vite](https://vitejs.dev/))
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **Styling:** Vanilla CSS
-   **State Management:** React Hooks (`useState`, `useEffect`)

## 📦 Setup & Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd todo-app
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
